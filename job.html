Develop an API using express and mongo

We are looking for a capable developer proficient with NodeJS to create an API for us that is somewhat similar to https://newsapi.org/.

Storage structure for articles
- id: string (unique)
- source: { id: string, name: string }
- author: { name: string, url: string }
- title: string
- description: string
- imageUrl: string
- publishedAt: date
- content: string
- keywords: array[string]
- category: string

API routes
- create article (API Level 1)
- update article by id (API Level 2)
- delete article (API Level 3)
- get article by id (API Level 1)
- get articles by category (API Level 1)
- get article from?date to?date (API Level 1)
- get articles by author (API Level 1)
- get articles by keyword (API Level 1)
- get articles by title (API Level 1)
